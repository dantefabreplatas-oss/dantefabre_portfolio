EJEMPLO 1

La empresa solicito: entregar un reporte para entender cómo la movilidad urbana (niveles de congestión, tiempos de viaje, retrasos) se relaciona con la productividad económica (PIB per cápita, desempleo) en las principales ciudades latinoamericanas.
El objetivo es identificar en qué ciudades invertir en infraestructura de transporte para aumentar la productividad y el bienestar de la población.

Para poder llegr al resultado tuve que trabajar en los Dataset para poder procesarlos y obtener las visualizaciones adecuadas. 

Entre las activides que tuve que realziar fueon las siguientes:

•	Conformé un DataSet único y limpio a partir de dos fuentes diferentes.
•	Aplique limpieza, estandarización y validación de tipos de datos.

Ya que los datos estaban estandarizados y adecuados para ser procesados empece con el procesado de los datos

•	Filtre y enfoque el análisis en ciudades latinoamericanas.
•	Calculé los indicadores agregados.
•	Realicé análisis exploratorios y visuales.

Al ir desarrollando las actividades, documenté todos los pasos en Jupyter Notebook,  para finalmente elaborar informe ejecutivo con hallazgos, implicaciones y reflexiones
Entre las habilidades requeridas para este proyecto, destacan: Jupyter Notebook | Python: pandas, numpy, seaborn, matplotlib 


Las preguntas a reponder a la empresa que nos pidio el analisis fueron:

¿Qué ciudades de América Latina presentan alta congestión y baja productividad económica?
¿Cuáles muestran los mejores indicadores combinados (movilidad eficiente y economía fuerte)?
¿Qué variables parecen tener una relación más fuerte con el desarrollo urbano?
